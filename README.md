# 👋 Xuanxuan Wu's Personal Homepage

My personal homepage built with Jekyll, featuring a cyber aurora theme with neon glow effects.

## 🌟 Features

- **Aurora Theme**: Beautiful gradient backgrounds with animated particles
- **Responsive Design**: Works seamlessly on desktop and mobile
- **Theme Switcher**: Toggle between Aurora, Cyberpunk, and Minimal themes
- **GitHub Integration**: Live contribution heatmap
- **Smooth Animations**: CSS animations and transitions throughout
- **Typewriter Effect**: Dynamic subtitle text

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/xuanxuanwuhh/xuanxuanwuhh.github.io.git

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

## 📁 Project Structure

```
├── _config.yml          # Site configuration
├── _layouts/            # Page layouts
│   ├── home.html        # Home page
│   ├── page.html        # Generic page
│   └── post.html        # Blog post
├── _posts/              # Blog posts
├── _includes/           # Reusable components
├── assets/css/          # Stylesheets
└── about.md             # About page
```

## ⚙️ Configuration

Edit `_config.yml` to customize your profile:

```yaml
title: Your Name
description: Your tagline
github_username: yourusername
email: your@email.com

author:
  name: Your Name
  bio: Your bio
  education: Your education
  location: Your location

projects:
  - name: Project Name
    description: Project description
    repo: https://github.com/...
    tags: [tag1, tag2]
```
